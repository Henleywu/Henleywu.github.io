我可以信任你吗

发表于 2020-05-08 分类于 胡咧咧 阅读次数： Valine：

最近中信银行泄露客户账户流水事件让许多人大跌眼镜。关注本身当然是好的，可能会推动整个行业加强对普通客户信息的保护。虽然我对这种可能抱有悲观态度。

之所以悲观，是因为这个事件能够曝光，本身就具有偶然性，事情败露的原因还是甲方一时失了智。东窗事发的概率太低，违反规则获得的收益又过于丰厚。如此有大前提有小前提，顺理成章地推导出“违反规则才是正确的”结论。利用人的道德、公司的品性来抵抗诱惑，我认为对大部分人来说是不现实的。作为万物之灵，人对于利益的计算是十分精明的。

今天又是非常高兴的一天呢，我已经可以用php给自己发邮件啦。

具体实现路径是在自己电脑上使用wamp server作为虚拟主机。然后使用vs code 写好php的内容。在这些都完成过后，还需要下载sendmile这个邮件系统。对php.ini和sendmail进行一番设置，因为需要指定smtp服务器和from 和 to 的邮箱来源，最后重启wamp server 在浏览器运行php就可以了。

两座“坑”开始的地方

发表于 2020-05-01 更新于 2020-05-02 分类于 文绉绉 阅读次数： Valine：

2020年的5月1日，雨。肝了一天终于把个人博客肝出来了。

最早是想用个人服务器来做博客的，做着做着发现，博客对我的吸引力太小。一是平常日记都不怎么写了，如何经营一个要时常更新的博客；二是性价比太低，现在视频更有市场。反正做着做着就觉得拿服务器做博客不现实，不经济。最终打动我的是因为这个博客不需要我的服务器，可以放在免费的平台上。而且在Hexo的帮助下，博客的页面设计和更新用少量代码就能完成，不会耗费太多精力。白嫖快乐。
